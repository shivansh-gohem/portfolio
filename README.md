# Shivansh | Frontend Engineer Portfolio

Welcome to my personal developer portfolio! This project showcases my skills, experience, and the interactive web applications I enjoy building. 

## 🚀 Features

- **Mario-Themed Interface**: A fully custom interactive, gamified design highlighting passion for creativity.
- **Tech Stack**: React, Vite, Tailwind CSS, TypeScript, and Shadcn UI.
- **Seamless Animations**: Smooth scroll and section reveals for top-tier user experience.

## 💻 Running Locally

To jump in and run this site locally:

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```
